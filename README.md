# Django Rest API
⭐ Please star if you found this helpful ⭐

Built an Instagram Backend MVP

using...

- Python (Django)
- PostgreSQL

## Includes
### - CRUD Functionality
Endpoints:
- /api/users/
- /api/profiles/
- /api/posts/
- /api/likes/
- /api/follows/
### - Basic JWT Authentication
Endpoints:
- /api/token/
- /api/token/refresh/
### - Tested with Postman ✅
